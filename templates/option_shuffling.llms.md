# Option Shuffling

Randomization

A template for creating questions with [shuffled options](../docs/survey-settings.llms.md#shuffled-questions).

Published

February 3, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "option_shuffling"
)
```
