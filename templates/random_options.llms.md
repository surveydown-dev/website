# Random Options

Randomization

A template for creating [randomized](../docs/randomization.llms.md#live-randomization) survey options.

Published

February 1, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "random_options"
)
```
