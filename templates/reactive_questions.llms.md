Reactivity

A template for creating [reactive](../docs/reactivity.llms.md) survey questions.

Published

March 1, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "reactive_questions"
)
```
