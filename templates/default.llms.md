# Default Template

Basic

A minimum template for starting from scratch.

Published

January 1, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey"
)
```

`template = "default"` is the default argument, so you do not need to specify it.
