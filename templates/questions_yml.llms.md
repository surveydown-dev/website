# Questions Defined with YAML

Basic

A template showcasing all built-in [question types](../docs/question-types.llms.md) using a YAML file.

Published

January 3, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "questions_yml"
)
```
