# External Redirect

Reactivity

A reactive question template with [redirectional](../docs/external-redirect.llms.md#reactive-redirect) links, accepting URL parameters.

Published

March 3, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "external_redirect"
)
```
