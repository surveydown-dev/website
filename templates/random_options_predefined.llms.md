# Predefined Random Options

Randomization

A template for creating [predefined randomized](../docs/randomization.llms.md#predefined-randomization) survey options.

Published

February 2, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "random_options_predefined"
)
```
