# Live Polling

Reactivity

A [reactive](../docs/reactivity.llms.md) question template of live-polling with responses shown as a bar chart.

Published

March 4, 2025

## Create this template

Run this command in your R console:

``` downlit
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "live_polling"
)
```
